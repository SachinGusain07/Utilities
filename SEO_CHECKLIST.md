# Toolnest - SEO & Deployment Checklist

## Before Going Live

### Technical Setup
- [ ] Replace `YOUR_ADSENSE_ID` in `app/layout.jsx`
- [ ] Replace `YOUR_GA_ID` in `app/layout.jsx`
- [ ] Update domain in metadata (change toolnest.app if different)
- [ ] Create custom favicon (favicon.ico, apple-touch-icon.png)
- [ ] Create OG image (og-image.png - 1200x630px)
- [ ] Test on mobile devices
- [ ] Test on desktop browsers
- [ ] Verify all tools work correctly
- [ ] Check forms and inputs
- [ ] Test copy to clipboard functionality
- [ ] Test file downloads
- [ ] Verify image uploads work
- [ ] Check calculator accuracy

### Performance
- [ ] Run Lighthouse audit (target: 90+)
- [ ] Check Core Web Vitals
- [ ] Optimize images
- [ ] Minify CSS/JS
- [ ] Enable GZIP compression
- [ ] Set up caching headers
- [ ] Test page load speed

### SEO
- [ ] Verify all meta tags
- [ ] Check keyword usage
- [ ] Review internal linking
- [ ] Test robots.txt
- [ ] Validate XML sitemap
- [ ] Check for broken links
- [ ] Verify canonical URLs
- [ ] Test structured data (JSON-LD)
- [ ] Check mobile responsiveness
- [ ] Test accessibility

### Security
- [ ] Enable HTTPS (Vercel does this)
- [ ] Set security headers
- [ ] Check for vulnerabilities
- [ ] Review CORS policies
- [ ] Test for XSS vulnerabilities
- [ ] Sanitize all user inputs

## Common Issues & Solutions

### Site Not Indexed
**Solution**: Submit sitemap to GSC, request indexing

### Low Rankings
**Solution**: Review keywords, improve content, build backlinks

### High Bounce Rate
**Solution**: Improve user experience, load speed, content clarity

### Low CTR
**Solution**: Improve title/meta description, test rich snippets

### No AdSense Earnings
**Solution**: Increase traffic, improve CTR, optimize ad placement
